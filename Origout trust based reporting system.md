It's for repositories not issues and activity inside them.
Has these items: 
1. Adult content
2. Copyright infringement 
3. Owner inactivity (inside repo isn't being moderated), will make repo read only until owner takes some kind of action (third time will make repo read only forever) 
4. Hate, discrimination, terrorism
5. Other illegal activity 

Uses can report repos, then volunteers can see an index of current repos that are reported using "origout report moderate":
